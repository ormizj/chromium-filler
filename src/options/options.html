    Two-step applications

    Postings that apply on the employer's own site are followed automatically, after any “before leaving” steps for that site (e.g. clicking its Save job button). Both the posting and the application it leads to are recorded.

    Review modal — size & position

    Drag the card to move it, drag its top-left corner to resize. The frame below is a screen to scale, so what you set here is what the modal gets on a real posting. This applies on desktop only — under 640px wide the modal is always a full-width bottom sheet, so on a phone the frame stands in for a desktop screen rather than for this one.

    Site configurations

    Per-site rules as JSON (array of configs). Detection, prep steps, extract selectors, field overrides, CV upload selector, submit-CV steps. Most sites are easier to set up visually — open a posting and use “Set up this site”.
