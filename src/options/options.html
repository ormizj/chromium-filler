Your profile

Values used to auto-fill forms. Leave blank to skip a field.

CV / Résumé

Stored on-device (chrome.storage.local). Attached to file inputs during fill.

Behavior

"Sent" is detected per site via that config's successSelector (a confirmation element) — the reliable signal. Without one, a plain form submit is used as a fallback (navigation flows only).

Two-step applications

Postings that apply on the employer's own site are followed automatically, after any “before leaving” steps for that site (e.g. clicking its Save job button). Both the posting and the application it leads to are recorded.

Site configurations

Per-site rules as JSON (array of configs). Detection, prep steps, extract selectors, field overrides, CV upload selector, submit-CV steps.

Job URL database

Paste a messy blob of text — every URL in it is extracted, normalized and deduped.
